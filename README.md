# debiantool
A debian tool written in python

### Installation:
```bash
git clone https://github.com/eris9/debiantool
cd debiantool
chmod +x debiantool
./debiantool
```
or:
Grab the latest binary from releases and:
```bash
chmod +x debaintool
./debiantool
```
